## Descripción
Ruta Cultural es una plataforma digital diseñada para conectar a turistas y residentes con la esencia gastronómica y artesanal de Calima El Darién. El proyecto busca eliminar intermediarios, permitiendo que los productores locales compartan su historia, sus productos y su ubicación de manera directa.
